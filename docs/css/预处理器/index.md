mixins,

## `@extend` 和 `@mixin` 的区别？

- `@mixin`: 混入，会做简单替换
- `@extend`: 会保证新的 class 和旧的 class，他们的表现完全相同

## 常见的CSS预处理器

- Sass: 2007, ruby
- Less: 2009, js

## 预处理器的作用

1. 更好组织CSS代码
2. 提高代码的复用性
3. 提高代码的可维护性
